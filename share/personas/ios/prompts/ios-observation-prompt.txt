/// Fetches the current balance for a Fun Card.
///
/// Use this method when you need the most up-to-date balance,
/// such as before processing a purchase or displaying the widget.
///
/// ## Overview
/// The balance is fetched from the server and cached locally.
/// Subsequent calls within 30 seconds return the cached value.
///
/// ## Example
/// ```swift
/// let balance = try await funCardService.fetchBalance(for: cardId)
/// balanceLabel.text = balance.formatted(.currency(code: "USD"))
/// ```
///
/// ## Error Handling
/// This method throws `FunCardError.networkUnavailable` if offline,
/// or `FunCardError.cardNotFound` if the card ID is invalid.
///
/// - Parameter cardId: The unique identifier of the Fun Card
/// - Returns: The current balance as a `Decimal`
/// - Throws: `FunCardError` if the balance cannot be fetched
func fetchBalance(for cardId: String) async throws -> Decimal
```
