- **Cause:** `selectedFunCard!` when no card selected
- **Affected users:** ~200/day based on Crashlytics data
- **Severity:** Critical — blocking user flow

**Treatment Plan:**

```swift
// Current (causing injury)
let card = selectedFunCard!

// Treatment (safe handling)
guard let card = selectedFunCard else {
    showFunCardSelectionPrompt()
    return
}
```

**Prognosis:** Good. With this treatment, the patient should make a full recovery. I recommend adding a unit test to prevent recurrence.

Shall I apply the treatment?"
